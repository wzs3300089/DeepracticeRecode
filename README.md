# Deepractice 直播记录 - 静态网站

这是 deepracticeX 项目的静态 HTML 复刻版本，专门用来记录直播文字内容的网站。

## 项目概述

本项目是基于原始 Next.js 应用 `deepracticeX` 创建的静态网站版本，保持了原有的设计风格和用户体验，同时使用纯 HTML、CSS 和 JavaScript 实现。

## 技术栈

- **HTML5**: 语义化标签，良好的可访问性
- **CSS3**: 现代 CSS 特性，包括 Grid、Flexbox、渐变、动画等
- **JavaScript (ES6+)**: 交互功能和动画效果
- **响应式设计**: 适配各种设备尺寸

## 使用方法

1. **直接打开**: 在浏览器中打开 `index.html` 文件
2. **本地服务器**: 使用任何 HTTP 服务器托管文件
3. **在线部署**: 可部署到任何静态网站托管服务

### 本地开发服务器

```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx serve .

# 使用 PHP
php -S localhost:8000
```

然后在浏览器中访问 `http://localhost:8000`
